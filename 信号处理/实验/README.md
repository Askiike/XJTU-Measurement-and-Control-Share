# 信号处理实验

本目录用于整理信号处理课程相关实验材料，包括 MATLAB 代码、实验说明、运行结果图和必要的示例数据。

## 当前内容

- `Signal_Processing/lab1/`：常见离散信号、信号基本运算、卷积运算、音频滤波示例；
- `Signal_Processing/lab2/`：时域采样与频域采样；
- `Signal_Processing/lab3/`：离散序列与模拟信号频谱分析；
- `Signal_Processing/lab4/`：数字滤波器设计相关实验。

## 使用说明

这些材料用于理解实验原理、复现实验流程、检查 MATLAB 环境和在已有基础上继续优化。请不要把代码或文档原样提交为自己的实验作业。

仓库是公开的。老师、助教和同学都可能看到同一份材料。更合理的使用方式是读懂代码、运行验证、根据自己的实验要求修改，并补充自己的分析。
